# comandos que utilice 

1. git init: para inicializar o crear el repositorio
2. git config --global user.email: para agregar mi correo al repositorio
3. "                  "user.name: para agregar mi nombre
4. git  commit . -m"" :para ir guardando los commit en el repositotio local con un mensaje agregado
5. git add . : este fue uno de los primeros que utilice para agregar los archivos al repositorio
6. git remote add origin: este comando lo utilice para conectar el repositorio local conn el que cree en git hub
7. git push -u origin master: este lo utilice para subir el contenido que tengo actualmente en el repositorio local al de la nube 
8. git log (este lo use para irar el historial de modificaciones de mi repositorio local)
